# Calls API endpoint 'DataStructure'

Calls API endpoint 'DataStructure'

## Usage

``` r
mt_data_structure(id)
```

## Arguments

- id:

  is a Dataset Series identifier.

## Value

a list with content of 'DataStructure' response.

## Examples

``` r
DOT <- mt_data_structure("DOT")
#> Data service did not respond.
```
