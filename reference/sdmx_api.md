# IMF SDMX 3.0 API

Low-level access to the IMF SDMX 3.0 structure, data, availability, and
metadata endpoints. These functions return the API response without
converting it to a tidy data frame. See
[`get_data()`](https://pedrobtz.github.io/imf.data/reference/get_data.md)
for the higher-level interface.
